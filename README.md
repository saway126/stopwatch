아이폰 17 Pro 디자인에서 영감을 받은 스톱워치 웹 앱입니다.

## 기능

- **시작 / 정지** — 타이머 제어
- **랩 기록** — 실행 중 현재 시간 저장
- **재설정** — 정지 후 전체 초기화
- **랩 목록** — 분할 시간(split)과 누적 시간 동시 표시
- **1/100초 단위** 정밀 측정

## 디자인

- 아이폰 17 Pro 프레임 (다이나믹 아일랜드 포함)
- 티타늄 사이드 프레임 광택 효과
- 주황색(`#ff8c00`) 포인트 컬러
- iOS 시계 앱과 동일한 UX (정지 시 랩 → 재설정 전환)

## 사용법

별도 설치 없이 `stopwatch.html` 파일을 브라우저로 열면 바로 실행됩니다.

```bash
# 저장소 클론
git clone https://github.com/saway126/stopwatch.git

# 파일 열기
open stopwatch/stopwatch.html
```

## 기술 스택

- CSS3 (애니메이션, 글래스모피즘)
